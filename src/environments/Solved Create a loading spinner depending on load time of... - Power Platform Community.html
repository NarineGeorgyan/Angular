            Hello!

             

            You can display a spinner depending while loading the data in a screen as follows:

             

            - First, in the tree view, select the screen where you want to have a loading spinner.

            - In the properties right panel, choose advanced

            - Under advanced, there is a property called "LoadingSpinner". By default is none but you can change it to LoadingSpinner.Data (shows the spinner while the data is loading) or to control (shows the spinner while a control is loading)
